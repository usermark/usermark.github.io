Frame

Label

可透過[‘text’]或config修改文字，config可修改UI屬性

Button

點擊事件有兩種設定方式：

  1. 透過command，只能處理左鍵單擊
  2. 透過bind，event.widget可取得原物件

Entry

修改文字有兩種設定方式

  1. 透過textvarialbe
  2. 透過insert，END為tkinter常數，表示結尾

ScrolledText

沒有textvariable屬性，只能用insert()修改文字

Bind

  • <Button-1> 左鍵點擊
  • <Button-2> windows和unix是中鍵點擊，mac是右鍵點擊，可用platform.system() == 'Darwin'判斷，Darwin為mac
  • <Button-3> 右鍵點擊
  • <Double-1> 左鍵連擊

lambda與閉包

閉包會把上一層的變數偷渡捕捉進來自己的function scope，Javascript便是用閉包來模擬class。

參考https://medium.com/citycoddee/python進階技巧-4-lambda-function-與-closure-之謎-7a385a35e1d8

舉例來說，當用for迴圈實作多個元件與按鈕，下面的寫法submit()永遠只會取到最後被指派的元件。

解法一，調整lambda寫法。
